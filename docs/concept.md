Corvin - нативное macOS-приложение (Swift + SwiftUI), превращающее голос в текст с помощью локальных моделей whisper.cpp

Ключевые характеристики:
- Полностью локальная работа — никакие данные не покидают компьютер
- Поддержка русского и английского с автоопределением языка
- Поддержка Apple Silicon (Metal) и Intel Mac (macOS 11.0+)

Приложение позволяет:
1. Использовать локальные модели whisper.cpp (ggml) для транскрибации речи в текст
2. Управлять моделями через встроенный менеджер — скачивание, удаление, переключение (tiny/base/small/medium/large-v3)
3. Получить полномочия универсального доступа (Accessibility API) для автоматической вставки текста в активное поле ввода
4. Активировать распознавание речи нажатием клавиши fn (настраиваемая)
5. Видеть статус записи и распознавания через floating-индикатор поверх всех окон
6. Просматривать историю всех транскрипций с поиском и группировкой по дням
7. Дублировать распознанный текст в clipboard (опционально)

Интерфейс:
- Menubar-иконка с dropdown-меню (последние записи, смена модели, настройки)
- Floating-индикатор (запись / распознавание / результат)
- Окно истории транскрипций
- Окно настроек (General, Models, Indicator, History, Permissions)
- Onboarding при первом запуске
